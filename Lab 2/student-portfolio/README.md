# Lab 2 – State Management and Routing in React

## Objective
Extend the Lab 1 portfolio by implementing React Router for client-side navigation and useState for dynamic UI updates.

## Features Implemented
- React Router v6 integration
- BrowserRouter setup
- Navigation using NavLink
- Home, Projects and Contact pages
- Controlled contact form using useState
- Live character counter
- Toggle help section
- Custom 404 Not Found page
- Responsive navigation with active route highlighting

## Technologies Used
- React
- React Router DOM
- JavaScript
- HTML
- CSS
- Vite
